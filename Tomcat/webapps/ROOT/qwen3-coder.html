什么是伯努利原理？

伯努利原理是流体力学中的一个基本定律，由瑞士科学家丹尼尔·伯努利于1738年提出。该原理描述了在理想流体（无粘性、不可压缩）的稳定流动中，流体的压强、速度和高度之间存在的关系。

简单来说，当流体在流动过程中速度增加时，其压强会减小；反之，当速度减小时，压强会增大。这就是著名的"伯努利效应"。

P + ½ρv² + ρgh = 常数

其中：

P = 流体压强

ρ = 流体密度

v = 流体速度

g = 重力加速度 (9.8 m/s²)

h = 高度

机翼升力原理

机翼上方空气流速快，压强小；下方空气流速慢，压强大。这种压强差产生向上的升力。

实际应用

✈️

飞机机翼

机翼的特殊形状使得上方空气流速快于下方，产生升力使飞机能够飞行。

⚽

足球弧线球

旋转的足球带动周围空气流动，一侧空气流速快压强小，另一侧流速慢压强大，产生侧向力使球弯曲。

噴射器

喷雾器

高速气流在喷嘴处产生低压区，将液体吸入并雾化喷出。

计算示例

假设水流在水平管道中流动，管道某处的截面积变为原来的一半。已知粗管处的流速为2 m/s，压强为1.5×10⁵ Pa，水的密度为1000 kg/m³。求细管处的流速和压强。

解题步骤：

1. 根据连续性方程 S₁v₁ = S₂v₂

因为 S₂ = S₁/2，所以 v₂ = 2v₁ = 2×2 = 4 m/s

2. 根据伯努利方程（水平管道，h₁ = h₂）：

P₁ + ½ρv₁² = P₂ + ½ρv₂²

3. 代入数值：

1.5×10⁵ + ½×1000×2² = P₂ + ½×1000×4²

1.5×10⁵ + 2000 = P₂ + 8000

P₂ = 1.5×10⁵ + 2000 - 8000 = 1.44×10⁵ Pa

因此，细管处的流速为4 m/s，压强为1.44×10⁵ Pa。
